# 🔐 Network Security Traffic Analyzer

## 📌 Overview

The **Network Security Traffic Analyzer** is a Python-based tool developed to analyze and visualize network traffic in order to identify potential security threats.

The system captures network packets from `.pcap` or `.pcapng` files, decodes packet information, and analyzes traffic to detect suspicious activities such as **Denial-of-Service (DoS) attacks, intrusion attempts, and abnormal traffic patterns**.

The application provides an interactive **Graphical User Interface (GUI)** that allows users to monitor network activity, filter traffic data, and visualize results through charts and statistical summaries.

---

## 🎯 Objectives

The main objectives of this project are:

* Develop a packet filtering and analysis tool using Python.
* Capture and decode network packets for both **IPv4 and IPv6 traffic**.
* Detect potential network security threats such as **DoS attacks, worms, and intrusion attempts**.
* Provide a **visual dashboard** to analyze traffic behavior and security risks. 

---

## ⚙️ System Architecture

The system processes network traffic through several stages:

1. Network Interface
2. Live Packet Capture using **Scapy**
3. Packet Decoding & Filtering
4. Threat Classification Engine
5. Graphical User Interface Visualization
6. Statistical Analysis and Data Export

The architecture allows users to monitor traffic patterns and detect anomalies efficiently. 

---

## ✨ Features

* 📂 Load `.pcap` and `.pcapng` files for analysis
* 📡 Decode packet information (IP, protocol, ports, packet size)
* 🔍 Filter traffic by:

  * IP version (IPv4 / IPv6)
  * Protocol (TCP, UDP, ICMP)
  * Network type
* 🚨 Detect network threats:

  * Denial-of-Service (DoS)
  * Intrusion attempts
  * Suspicious traffic
  * Malformed packets
* 📊 Traffic visualization charts:

  * Protocol distribution
  * IP version distribution
  * Traffic timeline
  * Security threat analysis
* 💾 Export analysis results to **JSON or CSV**
* 🖥️ GUI-based interface using **Tkinter**

## 📊 Example Analysis Results

The system is capable of analyzing large packet datasets. In one test, the analyzer processed **83,849 packets**, detecting multiple potential threats including DoS attempts and intrusion activities. 

Example insights generated by the system include:

* Protocol distribution (TCP, UDP, ICMP)
* IPv4 vs IPv6 traffic comparison
* Traffic timeline visualization
* Threat detection logs

These visualizations help administrators identify abnormal network behavior quickly.

---

## 🔒 Security Threat Detection

The system uses heuristic rules to detect potential threats such as:

* **DoS attacks** by identifying excessive SYN packets
* **Port scanning** activities
* **Suspicious access attempts** to sensitive ports (SSH, Telnet, RDP)
* **Malformed or abnormal packets**

Threats are highlighted using color-coded indicators in the GUI to make detection easier.
